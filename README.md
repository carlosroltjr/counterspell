# Counterspell

![Alt text](https://static.wikia.nocookie.net/dota2_gamepedia/images/9/90/Counterspell_icon.png/revision/latest?cb=20111005082151)
_Create an anti-newbie shell around you that blocks any malicious pick from the enemy team_

## Installation

```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
